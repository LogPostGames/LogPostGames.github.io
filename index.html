About Us

Located in Pittsburgh, Pennsylvania, Log Post Games is a new independent game development studio currently working on an unannounced narrative game. "We're" technically one person (Hi, I'm Jess!) making games, and I've been paid to make games for over 3 years now. Currently, I work at game studio, but Log Post Games is my passion project where I get to make all kinds of things I can enjoy and be proud of. I'm a programmer by trade, so a lot of the art you see will be made with some help but if it is ugly, well it is programmer art.
